# Urban Air Quality Correlation Engine

A Streamlit dashboard that explores relationships between:

- PM2.5 Air Pollution
- NO2 Air Pollution
- Traffic Volume
- Wind Speed
- Temperature

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy

1. Push project to GitHub
2. Login to Streamlit Community Cloud
3. Select repository
4. Deploy app.py

The application automatically generates a synthetic dataset if the CSV file is unavailable.
